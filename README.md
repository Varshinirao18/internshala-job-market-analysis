# Internshala Job Market Analysis using Web Scraping & EDA

## Project Overview
This project analyzes job listings scraped from Internshala to understand
salary trends, experience requirements, job roles, and city-wise demand
for fresh graduates.

## Data Source & Usage Disclaimer
This project uses data scraped from publicly available internship listings on Internshala for educational and learning purposes only.  
No personal, private, or restricted data was accessed or stored.

## Project Workflow
1. Sent HTTP requests to Internshala internship listing pages
2. Parsed HTML content using BeautifulSoup
3. Extracted internship details such as role, location, stipend, and skills
4. Cleaned and stored data in CSV format
5. Performed exploratory data analysis (EDA) using Python

## Key Findings
- Most internships are concentrated in metro cities
- Technical and development roles dominate listings
- Stipend varies significantly based on role and location
- Remote internships show increasing availability

## Objectives
- Identify high-demand job roles
- Analyze salary vs experience trends
- Compare salaries across cities
- Provide insights for fresh graduates
