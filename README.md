# Internshala Job Market Analysis using Web Scraping & EDA

## Project Overview
This project analyzes job listings scraped from Internshala to understand
salary trends, experience requirements, job roles, and city-wise demand
for fresh graduates.

## Objectives
- Identify high-demand job roles
- Analyze salary vs experience trends
- Compare salaries across cities
- Provide insights for fresh graduates

## Tools & Technologies
- Python
- BeautifulSoup
- Requests
- Pandas
- Matplotlib
- Seaborn

## Data Source
- Internshala Job Listings 

## Key Analysis
- Univariate, Bivariate, and Multivariate Analysis
- Salary vs Experience
- City-wise Salary Trends
- Job Role Trends
